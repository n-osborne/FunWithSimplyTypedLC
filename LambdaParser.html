data Ast Source #

Abstract Syntax Tree type declaration.

An Ast is either an empty tree (correspond to an empty program), or a Node with two children which are Abstract Syntax Trees with leaves. Nodes are for lambda application, so they have an arity of two. Leaves are for names.
